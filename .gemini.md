# Context Proiect: Family Expense Agent
**Branch actual:** teams/M3
**Tehnologii:** Java 21, Spring Boot, JPA, PostgreSQL, Lombok, LangChain4j.

## Reguli Stricte de Codare:
1. **Pachet de bază:** `com.proiect`
2. **Path API:** Toate controllerele încep cu `/v1/` (Exemplu: `/v1/expenses`). NU folosi prefixul `/api`.
3. **Baza de date:** Folosește `BigDecimal` pentru toate sumele de bani (`amount`).
4. **Testing:** Toate testele unitare trebuie să folosească profilul de test: `@ActiveProfiles("test")`.
5. **Entitate principală:** `ExpenseEntity` (id, amount, category, date).
